# SECURITY POLICY

## Reporting a Vulnerability

If you discover a vulnerability, please follow these steps:

1. Email us at: **security@example.org** (replace with your dedicated address)  
2. Use the subject line: **[SECURITY] <Project-Name>**  
3. Provide:
   - Affected repository/project name  
   - Description of the vulnerability  
   - Steps to reproduce (if possible)  
   - Your contact (optional) for follow-up  

We will respond within 72 hours and keep you updated.

## Supported Versions

The current supported version is: **v1.0.x**  
Any older versions may not receive security fixes.
