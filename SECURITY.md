# 🔐 Security Policy

## ✅ Supported Versions

以下のバージョンに対しては、現在もセキュリティアップデートが提供されています。

| バージョン | サポート状況         |
| ---------- | ------------------ |
| 5.1.x      | ✅ サポート中        |
| 5.0.x      | ❌ 終了              |
| 4.0.x      | ✅ サポート中        |
| < 4.0      | ❌ 完全終了          |

## 🛡 Reporting a Vulnerability

セキュリティ上の脆弱性を発見された場合は、以下の手順でご連絡ください：

1. メールまたは暗号化チャネルで報告してください  
2. 報告後 **48時間以内に初期応答** をいたします  
3. 対応状況・パッチ予定については、**継続的に進捗を通知**  
4. 重大なバグについては、**対応後に公式リリースノートに掲載**します  
5. 匿名報告、仮名報告にも対応可

📩 報告先メール（例）: `security@ayuchanen.org`
